# leo-ui

> 一个基于Vue.js的UI组件库

## Install

``` sh
npm install leo-ui --save
```

## Usage

```javascript
import Vue from 'vue'
import leo from 'leo-ui'
import 'leo-ui/lib/leo-ui.css'
Vue.use(leo)
```
